# ✍️ Kanji Practice — веб-приложение для изучения японской каллиграфии

**Kanji Practice** — это интерактивный тренажёр для изучения и практики написания японских иероглифов (кандзи). Приложение в реальном времени анализирует каждую черту, сравнивая её с эталоном из базы [KanjiVG](https://kanjivg.tagaini.net/), и даёт мгновенную обратную связь. Основная цель — помочь изучающим японский язык освоить правильный порядок, направление и форму черт, что критически важно для грамотного письма.

---

## 🚀 Возможности (текущая версия — Web MVP)

### ✅ Режим «По чертам»
- Пользователь последовательно рисует каждую черту иероглифа.
- После каждой черты приложение оценивает её по форме, длине и углу наклона.
- При оценке **≥ 80%** черта принимается и подсвечивается зелёным.
- Если оценка ниже — предлагается перерисовать (до трёх попыток).
- После трёх неудач черта помечается красным, приложение переходит к следующей.
- Эталон иероглифа может отображаться полупрозрачным слоем (включается/отключается кнопкой).

### 🧠 Умный анализ черт (Go-ядро)
- Алгоритм **Dynamic Time Warping (DTW)** для сравнения траектории пользователя с эталонной.
- Автоматическое центрирование и нормализация размера черт.
- Сравнение направления черты (угол наклона) с мягкими штрафами.
- Корректная обработка относительных и абсолютных координат SVG (парсер KanjiVG).

### 📱 Кроссплатформенное ядро на Go
- Основная логика распознавания и сравнения черт написана на **Go** в виде отдельного модуля (`kanji-recognition`).
- Модуль не зависит от UI и может быть переиспользован:
  - в веб-приложении (текущая версия),
  - в мобильном приложении (через `gomobile bind` для Android/iOS),
  - в десктопном приложении (через Fyne или Wails).
- Пакет `stroke` содержит:
  - `stroke.go` — базовые типы (`Point`, `Stroke`, `KanjiTemplate`, `StrokeResult`).
  - `comparator.go` — алгоритмы DTW, сравнения углов и длин, вычисление общей оценки.
  - `kvg_loader.go` — парсер SVG-файлов KanjiVG с интерполяцией кривых Безье и нормализацией направления.

### 🔌 API Endpoints

| Метод | Путь | Описание |
|-------|------|----------|
| `GET` | `/` | Отдаёт `index.html` |
| `GET` | `/api/kanji` | Список доступных иероглифов (по SVG-файлам) |
| `GET` | `/api/template/{id}` | SVG-файл эталона для отрисовки на canvas |
| `POST` | `/api/check-stroke` | Проверка одной нарисованной черты |

## 🔮 Планы по развитию

### 📝 Распознавание рукописных иероглифов (OCR)
- Интеграция **ONNX-модели** (PP-OCRv5 или аналогичной) для распознавания одиночных иероглифов.
- Пользователь рисует иероглиф целиком → система распознаёт его → автоматически загружает эталон для проверки.
- **Сценарий**: написание слов и вставка их в предложения.

### 📖 Чтение текстов
- Встроенная библиотека текстов с переводами для практики чтения.
- Возможность «написать слово из текста» — система проверит правильность написания.

### 🔄 Режим «Свободное рисование»
- Пользователь рисует иероглиф без подсказок.
- Система распознаёт его и сравнивает с эталоном (или предлагает правильный вариант).

### 📊 Прогресс и статистика
- Отслеживание изученных иероглифов.
- История попыток, процент успешных черт, «проблемные» иероглифы.

### 📱 Мобильное приложение
- Сборка нативного Android/iOS приложения через `gomobile bind`.
- Полноценное рисование пальцем/стилусом.
- Офлайн-режим (все эталоны и модель внутри приложения).

## 🎯 Ключевые особенности

1. **Обучение через практику** — приложение не просто тестирует, а ведёт пользователя по процессу написания.
2. **Объективная оценка** — каждая черта оценивается независимо по форме, длине и направлению.
3. **Терпимость к новичкам** — мягкие штрафы, три попытки на черту, понятная обратная связь.
4. **Модульная архитектура** — ядро можно переиспользовать в вебе, на мобилках и в десктопе.
5. **Открытые данные** — эталоны из KanjiVG (Creative Commons), модель из открытых источников.
